# Change Log

All notable changes to the "easy-typing-vscode" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## v0.1.0 2026-3-29

- 参照 Vim 插件的形式，优化了编辑的的稳定性。
- 添加了盘古格式化功能，可以通过命令 `Pangu Format` 在中文和英文之间补空格，并解决类似的排版格式问题。
- 重新组织了代码结构，添加了自动化测试样例。

## v0.0.12 2025-7-22

- 设置优化
- 根据目前的实际情况，更改激活语言为所有（*）
- 默认关闭调试信息输出
- 优化了规则（详见rules.ts）

## v0.0.1 2025-7-10

- Initial release
- 成对输入和删除中文符号功能
- 成对中文符号包裹功能
- 连输转换
- 删除空列表项
- 自定义激活语言
